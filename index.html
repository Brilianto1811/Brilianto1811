This application provides convenience in supporting office activities at the Communication and Informatics Office. This application can be used by all employees. The KOSMOS application has agenda features, employee assignments, and monthly reports of employee activities. The New Student Admissions System is a website used by admins and new students to facilitate the registration process on campus. This system has a new student admissions system flow menu, features for filling in new student data, printing student data and others.
